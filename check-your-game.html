A note on the set of words being considered

There are three different sets of words I considered using.

  1. Wordle's actual answers. This is the set of 2315 words that Wordle selects their answers from.
  2. Common 5-letter English words. These are the 4561 most common 5-letter words from the WordFreq dataset that are also considered valid entries by Wordle. The cutoff of 4561 words was choosen because it's the number of words which includes 95% of the Wordle's actual answers. Beyond that words get pretty obscure.
  3. All Wordle words. This set includes all 12972 5-letter words that Wordle accepts as a guess, most of which are really quite obscure.

This page defaults to "Common" words, because using Wordle's actual answers is kind of cheating, and because using all 13K 5-letter words includes a bunch of words that only scrabble fanatics know about.
