Exploring Wordle

Word Winnowing

You know that feeling in Wordle? When it goes from "it could be anything!" to "I can't think of anything it could be!" I was curious to see how that happens.

Here's a game of Wordle that was randomly generated when you opened this page. The answer is an actual Wordle answer, and all the guesses were selected from a set of all English words around as common as actual Wordle answers.

As you go through the game the number of possible answers remaining is shown by the size of the green circle, and the number of words you've eliminated is shown by the orange area around the green circle (only visible once you click the button below to get started).

Results in table form below
Word selection process

In these games, each guess is chosen randomly from all the possible remaining words that meet the criteria already learned (i.e. letters excluded, letters included, letter position/non-position, etc).

I'm not saying this is the optimal approach to playing Wordle, but an interesting one to look at none the less.

A note on words included for this page

There are three different sets of words I considered using.

  1. Wordle's actual answers. This is the set of 2315 words that Wordle selects their answers from.
  2. Common 5-letter words. These are the 4561 most common 5-letter words from the WordFreq dataset that are also considered valid entries by Wordle. The cutoff of 4561 words was chosen because it's the number of words which includes 95% of Wordle's actual answers. Beyond that words get pretty obscure.
  3. All Wordle words. This set includes all 12972 5-letter words that Wordle accepts as a guess, most of which are really quite obscure.

This page defaults to Wordle's actual answers for the answers, but to "common" words for the set of guesses, because using Wordle's actual answers is kind of cheating, and because using all 13K 5-letter words includes a bunch of words that mostly only scrabble fanatics know about.
